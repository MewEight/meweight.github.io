Battle Of The Rim
Prototype was made in Dota 2, to first test out the how the game would play. I made the call to do this because, it allowed us to quickly get the basic gameplay ready without worrying much about networking first.

Wrote a modifier system and base structure for the game in Unity which was inspired by how Dota 2 made their modding tools.

Create editor tools for designer to create skills easily.

Optimized input to send across networks, to make sure a better gaming experience.
